Hari Libur API

Static JSON API via GitHub Pages.

Endpoint

  • /api/{tahun} (JSON)
  • /api/{tahun}.json (JSON)
  • /api/index.json (daftar tahun tersedia)

Contoh

GET /api/2026

Format respons

{
    "year": 2026,
    "source": "https://kalenderku.id/2026",
    "scraped_at": "2026-04-22T03:37:56.841805+00:00",
    "holidays": [
        {
            "date": "2026-01-16",
            "name": "Isra Mikraj Nabi Muhammad SAW",
            "type": "Libur Nasional",
            "holiday_type": "libur"
        }
    ]
}
